# 🐱 Cat Tap

> **How high can your kitty fly?**

A addictive Flappy Bird-style game where you guide a cute cat through obstacles while collecting power-ups!

**🎮 Play Now:** https://spicyv-hub.github.io/CatTap/

---

## 🎯 How to Play

1. **Tap / Click / Spacebar** to make the cat jump
2. **Dodge** the red obstacles
3. **Collect** power-ups for special abilities
4. **Beat** your high score!

---

## ⚡ Power-Ups

| Power-up | Icon | Effect |
|----------|------|--------|
| **🛡️ Shield** | Blue orb | Survive one collision |
| **⏱️ Slow-mo** | Pink orb | Game slows down for 5 seconds |
| **⭐ Double Points** | Green orb | 2x score for 10 seconds |

---

## 🎮 Controls

| Input | Action |
|-------|--------|
| **Tap / Click** | Jump |
| **Spacebar** | Jump |
| **P** | Pause/Resume |

## 🚀 Quick Start

### Play Online (No Installation)
Visit: **https://spicyv-hub.github.io/CatTap/**

### Run Locally
```bash
# Using Python
cd cattap
python -m http.server 8000

# Then open http://localhost:8000
```

Or simply open `index.html` in your browser!

## 🎯 Tips & Tricks

1. **Short taps** = small jumps, **long holds** = same (it's tap-based)
2. **Stay centered** - gives you more reaction time
3. **Memorize patterns** - obstacles spawn at regular intervals
4. **Grab power-ups** - especially Shield for safety!
5. **Don't greedy** - play defensively for high scores
